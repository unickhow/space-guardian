# Space guardian

Enhance the readability of your code effortlessly with this Visual Studio Code extension. By intelligently adding spaces between Chinese and English characters, it provides mercy for your code's clarity and comprehension. Seamlessly integrating into your workflow, it supports various file formats including HTML, JavaScript, JSX, Vue, Svelte, TypeScript, TSX, and Markdown. ## How to use

`Cmd + Shift + P` (or `Ctrl + Shift + P` for Windows) and execute 'Make room', works on the current selection or the whole document.
